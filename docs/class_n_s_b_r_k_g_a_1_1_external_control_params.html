Detailed Description

Additional control parameters consumed by the calling application.

These parameters govern the outer loop orchestration (e.g., when to call exchangeElite(), pathRelink(), shake(), reset()) but are not used directly by the NSBRKGA engine. They are persisted to and from the same configuration file as NsbrkgaParams for convenience.

A value of 0 for any interval parameter disables the corresponding operation.

Detailed Description

Hamming distance between two chromosome vectors.

This functor discretises each allele into a bin index by computing \(\lfloor x \cdot B \rfloor\) where \(B\) is num_bins, and then counts the number of positions where the two chromosomes fall into different bins. The result is an unnormalised integer-valued distance in \([0, n]\) where \(n\) is the chromosome length.

Typical use: general-purpose chromosomes with continuous alleles that should be treated as approximately categorical after quantisation. The default num_bins = 2 splits \([0,1)\) into {< 0.5, >= 0.5}.

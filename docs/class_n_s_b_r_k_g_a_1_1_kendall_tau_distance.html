Detailed Description

Kendall Tau rank-correlation distance between two chromosomes.

This functor interprets each chromosome as a ranking of \(n\) items: the item at position \(i\) receives rank according to \(v[i]\). The distance is the number of concordance–discordance disagreements (inversions) between the two rank orderings — i.e., the number of pairs \((i, j)\) with \(i < j\) for which the relative order of item \(i\) and item \(j\) differs between the two chromosomes.

The result is unnormalised and lies in \([0, \binom{n}{2}]\). Complexity: \(O(n \log n + n^2)\).

Typical use: permutation-based decoders where allele rank order defines the solution, not the raw allele values.

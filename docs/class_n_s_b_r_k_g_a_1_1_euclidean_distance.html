Detailed Description

Euclidean ( \(L_2\)) distance between two chromosome vectors.

Computes \(\|v_1 - v_2\|_2 = \sqrt{\sum_i (v_1[i] - v_2[i])^2}\). The result is unnormalised; for chromosomes with alleles in \([0,1)\) the maximum value is \(\sqrt{n}\) where \(n\) is the chromosome length. Complexity: \(O(n)\).

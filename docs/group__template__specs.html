Template specializations for enum I/O.

Using slightly modified template class provided by Bradley Plohr (https://codereview.stackexchange.com/questions/14309/conversion-between-enum-and-string-in-c-class-header) we specialize that template to enums in the BRKGA namespace.

The EnumIO class helps to read and write enums from streams directly, saving time in coding custom solutions. Please, see third_part/enum_io.hpp for complete reference and examples.

Note
The specializations must be done in the global namespace.
Warning
The specialization must be inline-d to avoid multiple definitions issues across different modules. However, this can cause "inline" overflow, and compromise your code. If you include this header only once along with your code, it is safe to remove the inlines from the specializations. But, if this is not the case, you should move these specializations to a module you know is included only once, for instance, the main() module.

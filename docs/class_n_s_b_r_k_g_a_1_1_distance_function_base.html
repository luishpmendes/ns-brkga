Detailed Description

Abstract base class for chromosome distance functors.

Subclass this interface to provide a custom distance metric for path relinking. The distance is used to identify chromosome pairs that are sufficiently dissimilar to make path relinking worthwhile.

Invariants expected by the library:

  • distance(v, v) == 0 for any vector v.
  • distance(v1, v2) >= 0 for all v1, v2.
  • v1.size() == v2.size() at every call site; throw if this is violated.
